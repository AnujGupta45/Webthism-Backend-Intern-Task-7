Introduction

Welcome to the LuminaLead REST API Reference. All requests must use the base URL of the active server. The Content-Type header must be set to application/json. Response formats are always returned in standardized JSON payloads containing success indicators, messages, and payloads.

POST /api/leads/capture Submit Dynamic Form / Capture New Lead

This is the primary lead capture endpoint. It validates and sanitizes incoming fields using Zod and compiles submitted dynamic fields depending on form constraint profiles. It updates or creates CRM profiles based on unique email matches.

Request Body (JSON)

{
  "firstName": "Alex",        // String, Required (min 1 char)
  "lastName": "Russo",         // String, Optional
  "email": "alex@domain.com",  // String, Required (validated email format)
  "phone": "9876543210",       // String, Optional (validated 10-digit format)
  "campaignId": "uuid-string", // String, Optional (validated UUID)
  "formId": "uuid-string",     // String, Optional (validated UUID)
  "submittedData": {           // Object, Optional
    "companyName": "Acme Inc", // Custom fields depending on Form requirements
    "teamSize": 25
  }
}

GET /api/leads Retrieve Captured Leads Registry

Retrieves all captured leads. Supports optional query filtering by Campaign or Form ID.

Query Parameters

Parameter Type Description
campaignId String (UUID) Filter leads belonging to a specific campaign.
formId String (UUID) Filter leads captured via a specific form.
GET /api/contacts List Unified CRM Contacts

Retrieves the consolidated CRM contacts registry, displaying unified profiles sorted by latest sync update.

POST /api/campaigns Create Marketing Campaign

Creates a marketing campaign to organize forms and capture channels.

Request Body

{
  "name": "Social Media Ad Outreach", // String, Required (min 3 chars)
  "description": "Outreach from Facebook and Instagram ads.", // String, Optional
  "status": "ACTIVE" // Enum ("ACTIVE", "PAUSED", "COMPLETED"), Optional (default ACTIVE)
}
